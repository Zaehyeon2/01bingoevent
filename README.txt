TheKinGoD | 김재현
------------------------------------------------------------------------------------
Mac
Mac 폴더안에 있는 bingo.app을 실행시킨 뒤 조금 기다리시고 닉네임을 입력하면 됩니다.
확인되지 않은 개발자 에러가 뜰 경우 컨트롤을 누르면서 앱을 클릭후 열기 하시면 됩니다.
------------------------------------------------------------------------------------
Windows
Windows 폴더 안에 있는 bingo.exe를 실행시킨 뒤 조금 기다리시고 닉네임을 입력하면 됩니다.
